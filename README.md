# voila-example

## Data Description

한국 도로공사 공동데이텉 포털(http://data.ex.co.kr/)의 데이터 활용

## 필요조건

그림에 한글이 필요하기 때문에 "네이버"의 나눔바른고딕 TTF 설치후 진행([링크](https://hangeul.naver.com/font))

## 설치법: 

터미널에서

```
$ pip install numpy
$ pip install matplotlib
$ pip install pandas
$ pip install voila
```
